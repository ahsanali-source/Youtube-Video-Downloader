"YouTube Video Downloader"

About this project:
- This is a simple Python project that downloads YouTube videos using a video link. 
- Just run the program, paste the YouTube link, and the video will be downloaded.

Requirements:
1. Python installed
2. yt_dlp library

Editor:
-This project is created and tested using Visual Studio Code (VS Code).

How to Run:
1. Run the Python file
2. Paste the YouTube video link
3. Video will be downloaded in the same folder
